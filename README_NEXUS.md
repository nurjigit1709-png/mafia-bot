# 💰 Nexus Economy Bot

**Полнофункциональный экономический симулятор для Telegram**

## 🎮 Что это?

Nexus Economy - это игровой бот для Telegram-групп, который превращает обычный чат в экономическую игру. Пользователи могут:

✅ Работать и получать зарплату
✅ Создавать свой бизнес
✅ Покупать землю и недвижимость
✅ Торговать ресурсами (золото, серебро, руда)
✅ Соревноваться в рейтингах

## 🚀 Установка

```bash
git clone https://github.com/nurjigit1709-png/mafia-bot.git
cd mafia-bot

python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate  # Windows

pip install aiogram python-dotenv

python main_nexus.py
```

## 📋 Главные команды

- `/start` - Начать игру
- `/profile` - Мой профиль
- `/help` - Справка
- `/jobs` - Доступные вакансии
- `/mybusiness` - Мои бизнесы
- `/myland` - Моя земля
- `/market_gold` - Рынок золота
- `/top_balance` - Топ игроков

## 💼 Система работы

Каждый игрок может:
- Устроиться на работу в компании
- Получать зарплату 1 раз в 24 часа
- Уволиться (со штрафом 30% если контракт не закончен)

## 🏢 Создание бизнеса

Доступные типы бизнеса:
- 🏪 Маленький магазин (50K NC, 100м²)
- ☕ Кофейня (100K NC, 150м²)
- ⛏ Шахта (500K NC, 500м²)
- 🥈 Серебряная шахта (2.5M NC, 1000м²)
- 🥇 Золотая шахта (5M NC, 2000м²)

## 🌍 Земля

Цены на землю:
- 100м² = 20,000 NC
- 500м² = 90,000 NC
- 1000м² = 170,000 NC
- 5000м² = 800,000 NC

## 🛒 P2P Рынок

Игроки могут:
- Продавать и покупать золото, серебро, руду
- Торговать по свободным ценам
- Видеть рейтинг лотов по цене

## 🏆 Рейтинги

- 💰 Топ по балансу
- 🥇 Топ по золоту
- 🌍 Топ землевладельцев
- 🏢 Топ компаний

## 💡 Экономические правила

1. Деньги не должны появляться легко
2. Без работы или бизнеса дохода нет
3. Бизнес без рабочих не приносит прибыль
4. Золото получается только через добычу
5. P2P рынок - главный источник торговли
6. Нельзя уходить в минус
7. Все действия записываются в транзакции

## 🔒 Безопасность

- Проверка на отрицательные суммы
- Проверка на дублирующиеся покупки
- Максимальные лимиты на операции
- Логирование всех транзакций
- Система банов для читеров
